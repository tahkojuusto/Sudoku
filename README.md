#  Sudoku

A simple command-line interface (CLI) sudoku.

## Commands

Create a sudoku with ``n`` unknown numbers:
```
new n
```

Show the current sudoku game state:
```
rdr
```

Add the number ``z`` to the cell ``(x, y)``:
```
add x y z
```

Remove a number from the cell ``(x, y)``:
```
rem x y
```

Stop the game:
```
qit
```
